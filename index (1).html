Firebase nastavení — jednorázové

1. Vytvoř Firebase projekt

Jdi na console.firebase.google.com → "Add project" → zadej název → pokračuj bez Google Analytics → "Create project".

2. Přidej Realtime Database

V levém menu: Build → Realtime Database → "Create database" → vyber libovolný region → v Security rules zvol Start in test mode → "Enable".

3. Zkopíruj konfiguraci

Project Overview (ozubené kolo) → "Add app" → Web ikona (</>) → zadej přezdívku → "Register app". Zobrazí se blok firebaseConfig. Zkopíruj hodnoty do polí níže.

Konfigurace se uloží do localStorage tohoto prohlížeče.
